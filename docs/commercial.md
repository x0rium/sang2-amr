# Стратегия коммерциализации SANG2-AMR

## Позиционирование

**Tagline:** «Find what databases miss.»

SANG2-AMR — не замена CARD/ResFinder/RGI. Это **add-on слой**,
обнаруживающий аномальные структурные паттерны в геномах,
которые reference-based инструменты пропускают.

---

## Целевые сегменты (по приоритету)

### Segment 1: Референс-лаборатории (первый клиент)

**Кто:** Крупные клинические/национальные лаборатории, выполняющие WGS-типирование бактерий.
**Примеры:** PHE (UK), RKI (Германия), CDC (США), Роспотребнадзор (РФ), Quest Diagnostics, LabCorp, SYNLAB.
**Боль:** Получают изоляты с фенотипической резистентностью, но RGI/ResFinder не находит объяснение. Отчёт уходит как «mechanism unknown».
**Ценность:** SANG2-AMR предлагает структурных кандидатов → лаборатория может исследовать дальше.
**Модель:** SaaS / API. Интеграция в существующий NGS-пайплайн (Galaxy, Nextflow, Snakemake).
**Цена:** $2–10 за образец (маржинальная стоимость WGS ≈ $50–200, add-on в 5–10% от основной стоимости).

### Segment 2: Фармкомпании (R&D антибиотиков)

**Кто:** Команды, разрабатывающие новые антибиотики.
**Примеры:** Shionogi, Entasis (AstraZeneca), Paratek, Achaogen.
**Боль:** Нужно предсказать, какие механизмы резистентности появятся к их новому препарату.
**Ценность:** SANG2-AMR анализирует метагеномные данные из окружающей среды (сточные воды, почва) и находит «предрезистентность» — гены, которые потенциально могут мутировать в полноценную резистентность.
**Модель:** Лицензия + консалтинг. $50K–200K/год.

### Segment 3: Ветеринарные лаборатории

**Кто:** Лаборатории, мониторящие AMR в животноводстве.
**Примеры:** APHA (UK), BfR (Германия), USDA (США).
**Боль:** 70% антибиотиков используется в животноводстве. Регуляторы требуют мониторинг, но инструменты те же (reference-based).
**Ценность:** Те же функции, что Segment 1, но менее зарегулированный рынок (быстрее выход).
**Модель:** SaaS, аналогично Segment 1.

### Segment 4: Surveillance платформы

**Кто:** Глобальные сети мониторинга AMR.
**Примеры:** GLASS (WHO), EARS-Net (ECDC), NARMS (FDA/CDC), Pathogenwatch (Wellcome Sanger).
**Боль:** Работают на масштабе десятков тысяч геномов. Нужны автоматические алерты на «что-то новое».
**Ценность:** SANG2-AMR как real-time фильтр: «этот образец содержит структурно аномальный регион, не объяснимый CARD» → алерт.
**Модель:** Лицензия для платформы. $100K–500K/год.

## Регуляторная стратегия

### Фаза 1: Research Use Only (RUO)
- Без регуляторного одобрения
- Маркировка: «For research purposes only. Not for use in diagnostic procedures.»
- Достаточно для Segments 1–4 (все используют WGS в исследовательском контексте)
- Срок: немедленно после валидации

### Фаза 2: CE-IVD / FDA 510(k) (опционально)
- Только если есть спрос на клиническое применение
- Срок: 2–4 года после RUO
- Стоимость: $500K–2M
- Решение принимается после анализа спроса из Фазы 1

## Грантовое финансирование (параллельно)

### NIH R21 (Exploratory/Developmental Research)
- $275K на 2 года
- Идеально для «novel computational method for AMR detection»
- Подача: 3 цикла/год

### Wellcome Trust — Drug-Resistant Infections
- До £2.5M на 5 лет
- Фокус: новые подходы к AMR
- Подача: rolling

### JPIAMR (Joint Programming Initiative on AMR)
- Европейские коллаборативные гранты
- €200K–500K на 2–3 года

### BARDA (Biomedical Advanced Research and Development Authority)
- Контракты на AMR diagnostics
- $1M–50M (но конкурентные)
- Часть CARB-X accelerator

### РНФ / Минобрнауки (если РФ-юрисдикция)
- Грант РНФ: до 7 млн руб/год
- Направление: биобезопасность, антибиотикорезистентность

## Конкурентный ландшафт

| Игрок              | Подход              | Слабое место                          |
|--------------------|---------------------|---------------------------------------|
| CARD/RGI           | Reference + HMM     | Только известные гены                 |
| ResFinder          | BLAST               | Только известные гены                 |
| AMRFinderPlus      | HMM + BLAST         | Только известные гены + known SNPs    |
| Kover              | ML на k-мерах       | Black box, нужен training set         |
| DeepARG            | Deep learning       | Нужен training set, не интерпретируем |
| Seq2Geno2Pheno     | ML pipeline         | End-to-end, но reference-dependent    |
| **SANG2-AMR**      | **Ab initio signals** | **Нет зависимости от баз/обучения**  |

Ключевая дифференциация: все конкуренты требуют **либо базу известных генов, либо обучающую выборку**. SANG2-AMR — единственный подход, работающий ab initio.

## Риски

| Риск                                     | Митигация                                       |
|------------------------------------------|-------------------------------------------------|
| Сигналы не разделяют AMR от обычных генов | Validation phase: E2 dataset, leave-one-out     |
| Слишком много false positives            | Resonance geometric mean фильтрует шум          |
| Биологи не доверяют «чёрному ящику»      | Полная интерпретируемость: каждый сигнал объясним |
| Конкурент добавит ab initio модуль       | First-mover + публикация + open-source adoption |
| Регуляторные барьеры для клиники         | RUO first, CE-IVD только при подтверждённом спросе |
