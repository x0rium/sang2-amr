# Exploration Plan

## Последовательность экспериментов

Каждый эксперимент — отдельный файл. Результат фиксируется в самом файле.
В `src/` попадает только код, прошедший exploration.

---

### 001_alphabet_signals.py
**Цель:** Вычислить все 8 SANG2-сигналов на E. coli K-12 (U00096.3).
**Вход:** FASTA с хромосомой.
**Проверяем:**
- MI half-life ≈ 3 для кодирующих регионов
- Autocorrelation peak на lag=3
- ΔH отрицательный на промоторах/RBS
- H_ratio в диапазоне CODING (0.60–0.85 пересчитать для 4-буквенного алфавита)
- Hurst ≈ 0.6–0.7 (persistent)
**Критерий успеха:** MI half-life для CDS в диапазоне 2.5–4.0.

### 002_plasmid_contrast.py
**Цель:** Сравнить сигнальные профили хромосомы E. coli и плазмиды pKPC_CAV1321.
**Вход:** Два FASTA.
**Проверяем:**
- GC-content различается (Otsu разделяет)
- Hurst различается
- Regime classifier показывает разные распределения
- IS-элементы на плазмиде видны как regime shifts
**Критерий успеха:** ≥ 3 из 8 сигналов статистически различаются (p < 0.01).

### 003_pair_merge_hierarchy.py
**Цель:** Pair merge на нуклеотидной последовательности. Проверить иерархию.
**Вход:** E. coli K-12 (первые 100 kb для скорости).
**Проверяем:**
- Level 1 пары коррелируют с dinucleotide frequencies
- Level 2 единицы коррелируют с кодонами
- SDR на score-distribution даёт осмысленный порог
- Lossless reversibility (реконструкция = оригинал)
**Критерий успеха:** Lossless + уровень 2 содержит ≥ 50% стартовых кодонов (ATG).

### 004_resonance_motifs.py
**Цель:** Resonance scan на E. coli K-12. Найти мотивы.
**Вход:** E. coli K-12 (кодирующие регионы, 500 kb выборка).
**Проверяем:**
- Top-R пары содержат известные мотивы (TATAAT, AGGAGG)
- Eigendistances: k=3 peak (кодоны), k=10-12 (alpha-helix pitch в белках — indirect)
- CC vs SS resonance: аналог content/structure разделения
**Критерий успеха:** ≥ 3 из top-20 R-пар — биологически интерпретируемы.

### 005_amr_detection.py
**Цель:** Обнаружить известные гены AMR на плазмиде pKPC_CAV1321 ab initio.
**Вход:** pKPC_CAV1321 FASTA (без аннотации).
**Проверяем:**
- SANG2-AMR находит blaKPC регион как аномальный (cross-reference, regime shift, Hurst anomaly)
- Границы региона ± 500 bp от известной аннотации
**Критерий успеха:** blaKPC попадает в top-10 AMR-кандидатов.

### 006_mock_metagenome.py
**Цель:** Clustering контигов из HMP Mock Community без референса.
**Вход:** Собранные контиги из SRR2726667.
**Проверяем:**
- Кластеры соответствуют 20 видам (ARI > 0.5)
- MI half-life стабилен внутри кластера
**Критерий успеха:** ARI > 0.5, NMI > 0.6.

## Общие правила exploration

1. Каждый файл самодостаточен (можно запустить отдельно)
2. Фактический вывод фиксируется в комментариях / exploration/003_tools.md
3. Если результат противоречит ожиданиям — это ценнее, чем подтверждение
4. Exploration code НЕ оптимизируется. Читаемость > производительность
5. Все скачанные данные — в `exploration/data/` (gitignored)
